Instructions

This is a demo project for Ionic with adsense.
You can see how adsense can be integrated with Ionic Angular.

1. Overview

  1. Create your blank project with Ionic Angular.
  2. Install ng2-adsense via npm.
  3. Edit your project. Note that you need to add extra codes not mentioned in ng2-adsense page.
